Three artifacts in this folder, three ways to use them.

Static preview  →  preview.html

Opens immediately, no server needed. CSS animations run; click interactions don't.

Best for: a quick look at how the cloned page renders.

Real source code  →  clone/

The actual cloned files with their original names (HTML, CSS, JS, fonts). Edit, copy, build from these. Best viewed via the local server below — most modern frameworks (Next.js, React) won't fully run from file://.

Best for: using the clone as code.

Full interactive mode  →  start.bat / start.sh

Runs a tiny local web server, opens your browser, and serves the real source code. All animations and click interactions work.

Best for: the live page experience. Requires Node.js installed.
